## 安装

    1.安装 @vue/cli  如果存在 老版本的 vue-cli 需要删除
        Vue CLI 是下一代 CLI 工具，用于抽离构建应用程序的复杂性。当你用 Vue CLI 启动你的应用程序时，你还可以获得官方的 webpack 更新和配置更改和 Vuetify 的更新，而无需费力地升级程序。
    2.vue create 项目根目录名

    3. cd 项目根目录名
    4. vue add vuetify 或者 vue ui之后进入web可视化界面搜索vuetify进行安装
